◆ The Physical G29 Calibration Checklist

STEP 01
Hardware Lock

Turn G29 slide switch to "PS3" mode before plugging USB into Pi. Standardizes kernel input layout.

STEP 02
Verify Kernel

Run 'jstest /dev/input/js0' to check raw axis responses. Turn G29; Axis 0 should move from -32767 to 32767.

STEP 03
Deploy FFB

Execute generated 'ffset' command in bash. Autocenter springs instantly relax to emulated Isuzu D-Max EPS feel.

STEP 04
Diorama Test

Run 'mainapp_V4.py'. Ensure 1/2 G29 wheel turn safely locks Isuzu steering rack without rubbing tire liners.
